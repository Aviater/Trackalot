Author: Benedict Marien
Tags: location, user, ip, geo, coordinates, phone, telephone, city, country, track, data
Version: 1
Licence: GPLv2 or later

Plugin that allows you to create a widget from the 'Appearance' menu. 
When a user navigates a page where the widget is enabled it will display:

- The user's ip address.
- The user's location (city, region, country).
- The user's GeoCoordinates.
- The user's registered telephone number (if any).

Information to display can be modified when creating the widget.
